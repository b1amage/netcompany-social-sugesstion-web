# Commit feature

`feat: <description>`

Ex:

- feat: login form
- feat: post api for login

# Commit fix

`fix: <description>`

Ex:

- fix: login form ui
- fix: error calling api

# Commit chores

`chore: <description>`

Ex:

- chore: remove redundant space
- chore: format code

# Commit refactors

`refactor: <description>`

Ex:

- refactor: api by using axios client
